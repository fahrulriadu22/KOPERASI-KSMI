# KOPERASI KSMI

Aplikasi mobile Flutter untuk Koperasi KSMI.

## Memulai

Project ini adalah aplikasi Flutter untuk Koperasi Syirkah Muslim Indonesia.

## Persyaratan
Flutter SDK 3.0+

Dart 2.17+

Android Studio / VS Code

Device/Emulator dengan kamera

## Instalasi
### 1
Clone repository

git clone https://github.com/username/koperasi-fresh.git

cd koperasi-fresh

Install dependencies

### 2
install dependency
flutter pub get
Jalankan aplikasi

flutter run
